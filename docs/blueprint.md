# **App Name**: InterviewSense

## Core Features:

- User Authentication: Secure user registration, login, and session management using Firebase Authentication (Email/Google).
- Voice-Based AI Assistant: Speech-to-text input (Web Speech API) for user questions, processing by Gemini AI, and text-to-speech output of AI responses.
- Intelligent Response Tool: Generative AI (Gemini) that provides structured responses including explanations, sample answers/code, and tips for interview or programming questions.
- Interview Practice Mode: AI (Gemini) asks questions based on selected domains (e.g., Software Developer, HR), and analyzes user answers (text/voice).
- Conversation History & Profile: Store and display user questions and AI answers in Firestore, allowing users to revisit past interactions and manage their profile.
- Domain Selector Tool: Users can optionally select a specific domain (e.g., Programming, HR Interview) to guide the AI, or allow the AI to auto-detect the domain.

## Style Guidelines:

- Light color scheme emphasizing clarity and professionalism. Primary action color: #33808A (a serene teal-blue) representing knowledge and growth. Background color: #ECF5F7 (a very light, almost white, blue-green) for a clean canvas. Accent color: #26D9A9 (a vibrant turquoise-green) for highlights and calls-to-action.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-oriented feel. Body font: 'Inter' (sans-serif) for clean readability of explanations and tips.
- Use a set of consistent, line-based icons for key interactive elements like the microphone, chat bubbles, history, and user profile to maintain a modern and professional aesthetic.
- A responsive dashboard-style layout with distinct sections for the voice assistant interface, chat history, and interview practice mode, ensuring clear navigation and content prioritization.
- Subtle, fluid animations to provide feedback during voice input/output, AI processing, and content loading, enhancing the user experience without distraction.
